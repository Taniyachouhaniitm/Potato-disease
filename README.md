# Potato Disease Prediction App

A deep learning-based mobile application to detect potato leaf diseases using a CNN model. The model is built with TensorFlow, optimized with TensorFlow Lite, served via FastAPI, and integrated into a React Native app. Deployed on Google Cloud Platform (GCP).

---

## Features
- Predict disease from leaf images  
- CNN-based classification  
- Fast inference with TFLite  
- FastAPI backend  
- React Native mobile app  
- GCP deployment  

---

## Tech Stack
- TensorFlow / Keras  
- TensorFlow Lite  
- FastAPI  
- React Native  
- Google Cloud Platform
